#### โจทย์การบ้านที่ทำ

- Code ทั้งสองอันนี้ จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่

> for ( let i = 0; i < 5; i++ >)
> alert ( i )

// alert 0 ,1 ,2 ,3 ,4

> for ( let i = 0; i < 5; ++i )
> alert ( i )

// alert 0 ,1 ,2 ,3 ,4

    Code ทั้งสองชุดนี้ แสดงผล เหมือนกัน
