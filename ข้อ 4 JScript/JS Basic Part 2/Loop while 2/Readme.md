#### โจทย์การบ้านที่ทำ

- Code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่

> let i = 0;
>
> while ( ++i < 5> )
> alert ( i );

// alert 1 2 3 4

> let i = 0;
>
> while ( i++ < 5 >)
> alert ( i );

// alert 1 2 3 4 5

    Code ทั้ง 2 ชุด alert ไม่เหมือนกัน
